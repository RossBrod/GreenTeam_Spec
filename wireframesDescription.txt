Приложение состоит из 18 основных экранов. 
Цвета для оценки: 
-10 (очень плохо) / красный 
-5 / оранжевый
0 / желтый
5 / голубой
10 / зеленый


Экран со списком Бизнес Юнитов. Они обозначаются прямоугольниками разного размера. Размер зависит от ревенью и количества людей. Эти значения указаны на самом прямоугольнике с названием бизнес юнита. Так же каждый бизнес юнит имеет оценку по двум критериям: вверху справа острый квадрат - оценка производительности (берется из Jira). Оценка качества коммуникации обозначается квадратиком внизу справа с круглыми углами. В правом верхнем углу есть иконка человека -возможность перейти в личный кабинет сотрудника с правами Администратор. 

Нажимая на квадрат одного из бизнес юнитов можно перейти на аналогичный экран со списком клиентов. 

Еще один аналогичный экран, где можно увидеть список задач каждого клиента. При наведении (или нажатии правой кнопкой мыши) на одну из задач, ее можно закрепить на экране личного кабинета человека с ролью «Администратор». 

Экран личного кабинета
Здесь вначале вверху слева можно увидеть приветствие и имя владельца личного кабинета . Затем видна кнопка с Настройками. При нажатии на нее можно будет перейти к Экрану 9. Есть раздел Онлайн, где кружочки расположены в бегущей строке. Эти кружочки означают оценку коммуникации встреч, которые проходят прямо сейчас. При наведении (строка тогда должна замереть) или нажатии правой кнопкой мыши на кружочек, можно увидеть информацию о клиенте, названии проекта, кто является project manager и назначение встречи. Также эту встречу можно закрепить у себя на экране если присвоена роль «Администратор». В верхнем правом углу есть иконка выхода из личного кабинета. 
Под закрепленными встречами есть строка с уведомлениями о любом произошедшем изменении в проектах. К примеру : добавление нового члена команды, добавление звонка, добавление задачи и так далее. Справа есть счетчик уведомлений, где слева от знака / находится количество красных уведомлений, справа - общее количество.  
Фотографию владельца личного кабинета можно изменять, и возле нее так же присутствуют оценки по цветам за продуктивность(право сверху от фото, с острым углом) и качество коммуникации (право снизу от фото с круглым углом). 
Под фотографией содержится набор закрепленных для отслеживания проектов, имеющих красную оценку по коммуникации. 
Каждый проект также содержит информацию о своем названии, принадлежности к клиенту, срок дедлайна, имена hr менеджера и project manager. Каждый проект также имеет цветную оценку продуктивности и коммуникации.  
Правее раздела закрепленных для отслеживания проектов есть краткая информация о владельце личного кабинета:
Position
Role in the project 
Speciality
Work experience and level of expertise 
Courses taken
Также справа внизу располагается кнопка, перейдя по которой можно просмотреть отчеты по встречам, увидеть графики оценок встреч. 

Экран 5. 
Перейдя в один из закрепленных проектов можно попасть на экран с более подробной информацией об этом проекте. 
Справа вверху обозначен логотип клиента- заказчика и срок дедлайна проекта. 
А именно можно в левой части увидеть три активные кнопки:
Team
Project Tasks
Meeting

На прямоугольнике этих кнопок можно увидеть количество членов команды, задач по проекту, встреч соответственно. Встречи считающиеся не положительными так же отмечены числом рядом с общим числом встреч. 

Справа располагаются поля, где указана цель проекта, информация о заказчике и проекте, ожидаемый результат и важные заметки. Также есть возможность ознакомиться с отчетами о качестве встреч и прогрессе из Jira. 

Экран 6. 
Перейдя на предыдущем экране по кнопке «Team». Можно попасть на экран со списком членов команды. 
На карточке каждого из членов команды содержится:
Имя
Специальность
Как долго человек в проекте

Есть возможность написать сообщение на его почту
У пользователя с правами «Администратор» есть возможность удалить из команды человека. 
В правой части карточки можно увидеть:
Оценки трех последних анализов по продуктивности (сверху) и по качеству коммуникации (снизу). Цвет острого и круглого углов карточки указывает среднюю оценку члена команды. 
Развернув карточку можно получить в правой части экрана более подробную информацию о члене команды. А именно:
Position
Role in the project
Speciality 
Work experience and level of expertise 
Courses taken
Nudges (сюда записываются рекомендации по коммуникации сотрудника на встречах)

Также есть кнопка с графиком оценок коммуникации для отслеживания динамики и улучшений относительно пройденных курсов и nudges. 

В правом верхнем углу экрана есть кнопка для добавлением нового члена команды (может воспользоваться только пользователь роли Администратор. Он заполняет полностью информацию о новом сотруднике или она может подтягиваться с внутренних баз данных при переводе сотрудника с другого проекта. 
Так же здесь располагается кнопка перехода в gmail (или другой способ коммуникации членов команды). И логотип клиента, чьего проекта команда. 


Экран 7. 
С экрана 5 при нажатии на кнопку «Project Tasks» можно перейти на этот экран. 
Слева вверху находится кнопка перехода на страницу назад (то есть экран 5)
В правом верхнем углу находится логотип клиента, к чьему проекту относятся задачи. 
Есть еще иконка Jira (здесь может быть любой task manager)
Так же на экране располагаются фильтры задач
All
To do
In progress
Done

Под фильтрами расположены сами задачи (они появляются из Jira)
Карточка каждой задачи содержит ее название, человека, назначенного на ее выполнение, дедлайн. При нажатии (наведении) на карточку конкретной задачи можно увидеть ее расширенное описание. 


Экран 8
На этот экран можно попасть с экрана 5, нажав на кнопку «Meeting». Здесь можно увидеть список встреч. 
Карточка каждой встречи содержит информацию о:
Названии встречи
Списке участников
Ссылке
Дате
Времени

Так же присутствуют фильтры встречи: ALL, TO DO, DONE. Еще справа можно выбрать какие встречи отображать на экране: Общие или только те, где участвует владелец личного кабинета.

Справа вверху экрана можно увидеть иконку для перехода в Google календарь и логотип клиента, для чьего проекта открыт список встреч.
Так же нажав на стрелочку на карточке встреч можно увидеть расширенную информацию о встрече:
Classification 
Agenda (после того как встреча пройдет, здесь будет стоять оценка того, насколько содержание встречи соответствовало тому, что должно было на ней обсуждаться)
Estimated duration 
End time
Meeting materials (здесь будет прикреплена ссылка на виртуальный документ, в случае если перед встречей ее участникам потребуется ознакомиться с дополнительной информацией)
Assess readiness for meeting (здесь после ознакомления с дополнительной информацией к встрече по ссылке будет находиться тест из 5 вопросов, отвечая правильно на которые можно будет получить допуск к встрече)
Meeting analysis (здесь есть две иконки. нажимая на левую можно скачать анализ встречи. Если нажать на правую его можно будет просто прочитать на странице приложения)
Meeting format
Meeting record (здесь можно посмотреть запись встречи, также будут выделены для пересмотра части встречи, которые были оценены по коммуникации и выписана причина такой оценки)
Meeting rating (можно увидеть персональные рекомендации по прошествии встречи для владельца личного кабинета)



Экран 9. 
Сюда можно попасть с Экрана 4 при наличии роли Администратор. Этот экран принадлежит разделу настроек анализатора встреч по транскрипции разговора. Также есть возможность получать персональные рекомендации от анализатора для каждого участника встреч. Проверять соблюдение корпоративной культуры и повестки встречи. 
На этой странице слева содержатся кнопки перехода на экран 10 (Projects), экран 11 (AI Workers), экран 13 (Data), экран 16 (Settings). 
Также в основной части экрана содержатся два списка: Список проектов и Список AI Workers. 
По стрелке вверху справа списка проектов можно попасть на Экран 10. По стрелке вверху справа списка AI Workers можно перейти на Экран 11. Справа вверху экрана можно открыть раздел уведомлений, кликнув на колокольчик. И перейти в личный кабинет кликнув на кружочек с буквой имени владельца личного кабинета.


Экран 10. 
Сюда можно попасть с экрана 9, нажав на кнопку «Projects». Здесь находится список проектов с их описанием, которое включает: Name, Type, Created, Updated. На одной странице содержится по 10 проектов, их можно листать. Справа вверху есть кнопка «+», которая позволяет создать еще проекты. Справа вверху экрана можно открыть раздел уведомлений, кликнув на колокольчик. И перейти в личный кабинет кликнув на кружочек с буквой имени владельца личного кабинета.


Экран 11. 
Сюда можно попасть с главного экрана, нажав на кнопку «AI Workers». Здесь содержится список агентов-анализаторов. Можно просмотреть информацию по каждому агенту: Name, Created, Updated. Есть значок, позволяющий отредактировать то, как анализатор должен работать и по какой схеме будет выставлять оценки встреч и поведения людей. Справа вверху есть кнопка «+», позволяющая добавить новых агентов. Справа вверху экрана можно открыть раздел уведомлений, кликнув на колокольчик. И перейти в личный кабинет кликнув на кружочек с буквой имени владельца личного кабинета.


Экран 12. 
Сюда можно попасть с экрана 11. Здесь находится чат с агентом-анализатором. Сюда попадают транскрипции диалогов на встречах. В качестве ответа можно получить анализ по коммуникации. Справа в разделе “Instructions” находится список Знаний. Справа вверху экрана можно открыть раздел уведомлений, кликнув на колокольчик. И перейти в личный кабинет кликнув на кружочек с буквой имени владельца личного кабинета.


Экран 13. 
Сюда можно перейти нажав кнопку «Data» на экране «Домой». Здесь содержится список инструкций, которые содержатся внутри агента и непосредственно анализируют встречи и производят выставления оценок по коммуникации. Справа вверху экрана можно открыть раздел уведомлений, кликнув на колокольчик. И перейти в личный кабинет кликнув на кружочек с буквой имени владельца личного кабинета.


Экран 14. 
Сюда можно попасть с экрана 13. Здесь находится описание того, как агенты должны выполнять анализ встречи и производить оценку качества коммуникации сотрудников, иначе Список Инструкций. Справа вверху экрана можно открыть раздел уведомлений, кликнув на колокольчик. А также перейти в личный кабинет кликнув на кружочек с буквой имени владельца личного кабинета. Два ключика находятся здесь по причине того, чтобы для того что инструкция была сохранена отредактированной и могла применяться, она должна получить одобрение от Head of HR и Head of PM. 



Экран 15. 
Сюда можно попасть с экрана 13, перейдя в раздел «Knowledge». Здесь содержится список Знаний. Его нельзя редактировать. Можно только добавлять новые. Справа вверху экрана можно открыть раздел уведомлений, кликнув на колокольчик. И перейти в личный кабинет кликнув на кружочек с буквой имени владельца личного кабинета.



Экран 16. 
Сюда можно попасть с главного экрана 10. Здесь находится список настроек организации. Справа вверху экрана можно открыть раздел уведомлений, кликнув на колокольчик. И перейти в личный кабинет кликнув на кружочек с буквой имени владельца личного кабинета.



Экран 17. 
Сюда можно перейти с экрана 11, нажав на кнопку «+». Эта страница позволяет добавить агентов и инструкции. Справа вверху экрана можно открыть раздел уведомлений, кликнув на колокольчик. И перейти в личный кабинет кликнув на кружочек с буквой имени владельца личного кабинета.
